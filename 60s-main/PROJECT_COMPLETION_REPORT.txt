🚀 API 功能演示
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

端点 1: GET /power/daily?encoding=json
└─ 返回今日电力日报，包含 5-10 条资讯
└─ 格式: JSON
└─ 示例响应: 200 OK，包含日期、标题和新闻列表

端点 2: GET /power/daily?encoding=text
└─ 返回纯文本格式的电力日报
└─ 格式: 纯文本（UTF-8）
└─ 示例响应: 200 OK，包含格式化的文本内容

端点 3: GET /power/news?category=energy&limit=10
└─ 按分类返回资讯列表
└─ 支持分类: all, energy, policy, market 或具体标签
└─ 示例响应: 200 OK，包含指定分类的资讯

端点 4: GET /health
└─ 健康检查端点
└─ 示例响应: 200 OK {"status":"ok","service":"power-api",...}

端点 5: GET /
└─ API 信息端点
└─ 示例响应: 200 OK，包含 API 版本、端点列表等
